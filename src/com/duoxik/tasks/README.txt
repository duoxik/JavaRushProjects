agregator - парсер сайтов по поиску работы (headhunter.ru, moikrug.ru). Поиск вакансий java программиста и преобразование их в общий html документ в отформатированном виде. Паттерны и технологии: MVC, Strategy, Jsoup.

archivator - обычный архиватор. Архивирование и разархивирование файла, добавление и удаление файла в(из) архив(а). Паттерны: Command

armstrong - поиск чисел Армстронга до заданного значения. Решение построено на алгоритме, который позволяет найти все числа армстронга в диапозоне [0, LONG.MAX_VALUE] за 4-5 секунд без использание thread'ов, за 1.5-2 секунды с использанием thread'ов.

calculator - первая моя версия калькулятора(не очень удачная). Парсинг математических выражений основанный на рекусрсивном алгоритме поиска скобочных выражение и их вычисления. 

calculatorV2 - вторая версия калькулятора математических выражений (более удачная). Основан на том же принципе. Более аккуртный код + добавлена возможность использовать тригонометрические функции (sin, cos, tan). Также присутствуют JUnit тесты.

game2048 - полноценная игра 2048. Архитектура программы основана на паттерне MVС. Присутствует пользовательский интерфейс. Паттерны и технологии: MVC, Swing.

gamespacewar - консольная игра SpaceWar. 

htmleditor - HTML редактор с пользовательским интерфейсом, который позволяет конвертировать обычный текст в HTML и обратно. Присутствует возможность подгружать файлы в редактор, форматировать их. Технологии: Swing, JSoup.

logparser - обычный парсер логгов, с возможностью написания запроса типа SQL (баловство не более).

patterns - папка в которую планировал скидывать примеры реализаций используемых патернов.

restaurant - консольное приложение моделирующие работу ресторана. Когда клиент определяется с заказом, он отправляется свободному повару. Во время приготовления клиенту показывается реклама. Также ведется статистика доходов, в отдельном классе. Паттерны: Observer, Singleton.

shortener - консольное приложение укорачиватель ссылок. Присутсвует разновидность хранение данных в различных структурах данных, проводится сравнение производительности в тестовых папках. Паттерны и технологии: Strategy, Apache Commons Collections, Google Common Collectinos (Guava)

tetris - консольная игра тетрис.

tree - реализация красно-черного дерева.
